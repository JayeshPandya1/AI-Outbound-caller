Quick Setup Guide

  1. Go to Settings and enter your API keys for LiveKit, Gemini, and Vobiz.
  2. Click Save on each section.
  3. Click ⚡ Auto-Create SIP Trunk in the Vobiz section (checks keys and links Vobiz to LiveKit).
  4. Check Global AI Prompt — customize the instructions to fit your business name and service.
  5. (Optional) In Agents, create a profile choosing a preferred Voice (Aoede is recommended) and make it the default.
  6. Run a test call via Single Call to your own personal phone number to ensure audio flows and SIP routes correctly (expect up to ~20 seconds of ringing).
